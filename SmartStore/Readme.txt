Project Specification
##########################################

This is a retail Application used for the following.

1. Listing products (currently listing from a locale file "/sample_data.json")
2. Searching, categorising Products
3. Adding Products to cart


Can be added the following Features or Enhancements
##########################################

1. Provide an option to the user to select the quantities
2. Product reviews.
3. My Account section and Histories
4. Showing availabilities of products or Adds according to previous shopping history or user's taste or choices.

How to Run this Application
##########################################

Though we need CocoaPod to build the App freshly, I have included all the pod files.
Running "pod install" is not necessary.
